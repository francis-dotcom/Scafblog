## Minimum Laptop Specifications

### Hardware Requirements (Non-Negotiable)

**CPU**
- Intel Core i7 11th Gen or newer (8+ cores)
- OR AMD Ryzen 7 5000 series or newer (8+ cores)

**RAM**
- 32GB minimum
- 64GB recommended for Modules 2-7

**Storage**
- 1TB NVMe SSD minimum
- 2TB recommended for datasets and VMs

**GPU (Critical for AI Modules)**
- NVIDIA RTX 3060 8GB minimum
- RTX 4060/4070 12GB+ recommended
- AMD GPUs not supported for CUDA-dependent labs

**Display**
- 1920x1080 resolution minimum
- 1440p+ recommended for multitasking

**Battery & Cooling**
- 6+ hour battery capacity (for unplugged sessions)
- Adequate thermal design (no throttling under load)

## Required Software Stack

### 1. Virtualization Layer (Install Before Course)
```bash
sudo apt update
sudo apt install qemu-kvm libvirt-daemon-system libvirt-clients
sudo apt install virtualbox-7.0 virt-manager
sudo systemctl enable libvirtd
```

### 2. Containerization
```bash
sudo apt install docker.io docker-compose podman
sudo usermod -aG docker $USER
sudo systemctl enable docker
```

### 3. AI/ML Development Stack
```bash
# Python environment
python3 -m venv ~/aisec
source ~/aisec/bin/activate

# Core ML libraries
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
pip install tensorflow-gpu jax jaxlib
pip install transformers datasets accelerate
pip install openai anthropic langchain
```

### 4. Security Tooling
```bash
# Offensive security tools
sudo apt install nmap burpsuite wireshark metasploit-framework
sudo apt install john hashcat sqlmap

# Development and analysis
pip install scapy impacket pwntools requests beautifulsoup4
```

### 5. Development Environment
```bash
# Code editors and version control
sudo snap install code --classic  # VS Code
sudo apt install neovim git gh

# Terminal and productivity
sudo apt install tmux zsh terminator htop
```
